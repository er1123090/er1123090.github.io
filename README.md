# Minseo (Eric) Kim

Static academic personal website for GitHub Pages.

## Local Preview

From this directory:

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```

Opening `index.html` directly also works for a quick check.

## GitHub Pages

For a user site, create a repository named:

```text
er1123090.github.io
```

Publish from the repository root on the default branch. GitHub Pages should serve the site at:

```text
https://er1123090.github.io/
```
